토리펀 (메이플키우기 최종데미지 계산기)
스카니아9 히수토리
· 모든 % 수치는 그대로 입력합니다. 예) 크리티컬 데미지 172.3% → 172.3, 스킬데미지 58% → 58
· 아래 “증가/감소” 칸에 + / − 값만 넣으면 신규 수치와 최종 데미지가 자동 계산됩니다.
· 예: 크확이 0.7% 증가라면 +0.7, 크뎀이 5.5% 감소라면 -5.5 입력
· 스탯비례 데미지는 주스탯 % 증가량에 따라 달라지므로, 착용 전/후 값을 계산해 증감만 직접 입력하세요.
  예시: 착용 전 400% / 착용 후 300% → -100
