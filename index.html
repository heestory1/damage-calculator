12/1 업데이트 · 공격력 계산 로직 보정 · 효율 분석 안정화
· 모든 % 수치는 그대로 입력합니다. 예) 크리티컬 데미지 172.3% → 172.3, 스킬데미지 58% → 58
· 증가/감소 칸에는 + / − 변화량만 입력합니다. 예) 크확 +0.7% → +0.7, 크뎀 -5.5% → -5.5
· 상태창 공격력은 마을 기준 그대로 입력 (무기 보유효과 포함 / 스킬 제외)
· 공격력 증감란에는 순수 변화량만 입력합니다.
· 증감을 비우면 신규 = 기존 수치로 자동 유지됩니다.
· 스탯비례 데미지는 착용 전/후 차이만 직접 계산하여 입력
예: 400% → 300% = -100
· 무기 보유효과(합연산)는 이미 반영된 값으로 보고 장비 변화량만 입력
· 스킬 공격력%(합연산)에는 합산된 증가량 입력
세팅 저장 / 불러오기
능력치 입력
