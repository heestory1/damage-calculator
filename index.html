최종 데미지 비교 계산기

스카니아9 히수토리
· 모든 % 수치는 그대로 입력합니다. 예) 크리티컬 데미지 172.3% → 172.3, 스킬데미지 58% → 58
· 아래 “증가/감소” 칸에 + / − 값만 넣으면 신규 수치와 최종 데미지가 자동 계산됩니다.
· 예: 크확이 0.7% 증가라면 +0.7, 크뎀이 5.5% 감소라면 -5.5 입력 (증가는 빨간색, 감소는 파란색으로 표시).

능력치 입력

항목 기존 수치 증가/감소 (신규 - 기존) 신규 수치 (자동)

세팅 저장 / 불러오기

· 세팅에는 기존 수치 + 증가/감소 값이 함께 저장됩니다.
· 이 기기/브라우저에만 저장되며, 쿠키/저장소 삭제 시 함께 사라집니다.
