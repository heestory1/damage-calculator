12/1 업데이트: 공격력 계산 로직 보정 · 효율 분석 안정화
· 모든 % 수치는 그대로 입력합니다. 예) 크리티컬 데미지 172.3% → 172.3, 스킬데미지 58% → 58
· 증가/감소 칸에는 + / − 변화량만 입력합니다. 예) 크확 +0.7% → +0.7, 크뎀 -5.5% → -5.5
· 상태창 공격력 칸에는 마을에서 보이는 상태창 공격력을 그대로 입력합니다. (무기 보유효과 포함, 스킬 미포함)
· 공격력 증가/감소 칸에는 무기보유효과/스킬 적용 전의 순수 변화량만 입력합니다.
· 증감을 비워두면 신규 수치 = 기존 수치로 자동 복사됩니다.
· 스탯비례 데미지는 주스탯 % 증가량에 따라 달라지므로, 착용 전/후 스탯비례%를 직접 계산하여 증감만 넣어 주세요.
  예시: 착용 전 400% / 착용 후 300% → 증감 칸에 -100 입력
· 무기 보유효과(합연산/직접 계산기로 더해야함)는 상대적 공격력 값을 이미 올려준 상태로 생각하고, 장비로 인한 순수 변화량만 입력하세요.
· 스킬 공격력% 증가(합연산)에는 스킬/버프로 인해 합산된 공격력% 증가량을 입력해 주세요.
